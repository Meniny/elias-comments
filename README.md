# Comment package for Atom ![Build Status](http://meniny.cn/img/build_passing.svg)

An Atom package for inserting comments with keyboard shortcut.

## Usage

Hit `shift + cmd + M` after text selection.

## Language Support

- .class
- .coffee
- .cs
- .css
- .html/.htm
- .js
- .less
- .md
- .rb
- .yml
- .php
- .py
- ...
